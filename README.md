# GetGravatar

Returns a Gravatar image for a given user. Requires valid name and email user attributes.

## Installation

Add this line to your application's Gemfile:

    gem 'get_gravatar'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install get_gravatar

## Usage

`gravatar_for(user)`
